# SALES PREDICTION USING PYTHON 📊

## Project Overview

This project is part of the **CODSOFT Data Science Internship**.
The objective of this project is to build a machine learning model that can predict product sales based on advertising expenditure across different platforms.

Sales prediction helps businesses estimate future sales and make better marketing and financial decisions.

---

## Problem Statement

Sales prediction involves forecasting the amount of a product that customers will purchase by analyzing factors such as:

* Advertising budget
* Marketing platforms
* Customer reach

Using machine learning techniques, we can analyze historical advertising data and predict future sales.

---

## Dataset

The dataset used in this project contains advertising spending on different platforms:

* TV Advertising
* Radio Advertising
* Newspaper Advertising

The target variable is:

* Sales

The model learns the relationship between advertising expenditure and sales.

---

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Google Colab / Jupyter Notebook

---

## Machine Learning Model

In this project, we used **Linear Regression** to build the prediction model.

Steps involved in the project:

1. Importing required libraries
2. Loading the dataset
3. Data preprocessing
4. Splitting the dataset into training and testing sets
5. Training the machine learning model
6. Predicting sales values
7. Evaluating model performance



## Results

The model predicts sales based on advertising expenditure.
This helps businesses understand how different advertising channels influence product sales.

---

## Conclusion

Sales prediction using machine learning helps companies:

* Understand the impact of advertising
* Optimize marketing strategies
* Improve decision-making
* Increase business revenue

This project demonstrates how machine learning can be used to analyze marketing data and predict sales effectively.
