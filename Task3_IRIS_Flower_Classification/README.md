# IRIS FLOWER CLASSIFICATION 🌸

## Project Overview

This project is part of the **CODSOFT Data Science Internship**.
The goal of this project is to build a machine learning model that can classify iris flowers into different species based on their sepal and petal measurements.

The Iris dataset contains three species of flowers:

* Setosa
* Versicolor
* Virginica

Using machine learning techniques, the model learns patterns from the dataset and predicts the species of a flower.

---

## Dataset

The dataset used in this project contains the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species (Target Variable)

These measurements help the model identify the correct flower species.

## Machine Learning Model

In this project, we used the **Logistic Regression** algorithm to train the classification model.

The process includes:

1. Loading the dataset
2. Data preprocessing
3. Splitting the dataset into training and testing sets
4. Training the machine learning model
5. Predicting the species of iris flowers
6. Evaluating the model accuracy

---

## Model Accuracy

The model achieved an accuracy of approximately **90% – 97%**, which shows that the model performs well in classifying iris flowers.
